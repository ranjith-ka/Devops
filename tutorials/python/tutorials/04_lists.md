# Creating & Accessing Lists

```python
my_list = [1, 2, 3, "hello"]
```

- **Indexing:**
  ```python
  my_list[0]      # First item
  my_list[-1]     # Last item
  ```

- **Slicing:**
  ```python
  my_list[1:3]    # Items at index 1 and 2
  my_list[::-1]   # Reversed list
  ```

---

# Adding & Removing Items

- **Add:**
  `append()`, `extend()`, `insert()`

- **Remove:**
  `remove()`, `pop()`, `clear()`

---

# Finding & Checking

- **Membership:**
  ```python
  "hello" in my_list
  ```

- **Find index/count:**
  `index()`, `count()`

---

# Changing Lists

- **Assigning:**
  ```python
  my_list[0] = 100
  ```

- **Slicing assignment:**
  ```python
  my_list[1:3] = [8, 9]
  ```

---

# Sorting & Reversing

- `sort()`, `sorted()`, `reverse()`
