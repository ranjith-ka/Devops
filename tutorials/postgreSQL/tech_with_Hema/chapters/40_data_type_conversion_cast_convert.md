## 40. Data Type Conversion (CAST / CONVERT)

[▶ Watch this section](https://www.youtube.com/watch?v=btZdKO17xOM&t=29568s) · [⬆ Back to top](#table-of-contents)

Convert between data types safely.

```sql
SELECT CAST(salary AS VARCHAR(20)) FROM employees;
SELECT CONVERT(VARCHAR, GETDATE(), 103);  -- dd/mm/yyyy format
```
