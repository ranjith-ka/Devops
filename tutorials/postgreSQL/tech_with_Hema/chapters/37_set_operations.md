## 37. Set Operations

[▶ Watch this section](https://www.youtube.com/watch?v=btZdKO17xOM&t=24973s) · [⬆ Back to top](#table-of-contents)

Combine results from multiple SELECT statements.

```sql
SELECT name FROM employees_2024
UNION
SELECT name FROM employees_2025;   -- distinct combined

-- Also: UNION ALL, INTERSECT, EXCEPT
```
