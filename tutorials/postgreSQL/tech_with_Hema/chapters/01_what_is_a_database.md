## 1. What is a Database

[▶ Watch this section](https://www.youtube.com/watch?v=btZdKO17xOM&t=0s) · [⬆ Back to top](#table-of-contents)

A **database** is an organized collection of data stored electronically so it can be created, read, updated, and deleted efficiently.

**Why we need databases:**
- Store large volumes of data reliably.
- Retrieve specific data quickly.
- Prevent duplication and inconsistency.
- Support many users and applications at the same time.

Databases power almost every application — banking, e-commerce, hospitals, student portals, and more.
