## 11. UNIQUE Constraint

[▶ Watch this section](https://www.youtube.com/watch?v=btZdKO17xOM&t=4689s) · [⬆ Back to top](#table-of-contents)

Prevents duplicate values in a column (can still allow one NULL).

```sql
CREATE TABLE users (
    id INT PRIMARY KEY,
    email VARCHAR(255) UNIQUE
);
```
