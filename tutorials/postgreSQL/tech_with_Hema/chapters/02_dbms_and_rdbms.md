## 2. DBMS and RDBMS

[▶ Watch this section](https://www.youtube.com/watch?v=btZdKO17xOM&t=148s) · [⬆ Back to top](#table-of-contents)

- **DBMS** (Database Management System) — software that manages databases (e.g. file-based systems).
- **RDBMS** (Relational DBMS) — stores data in **tables** with rows and columns and supports **relationships** between tables.
- Examples of RDBMS: SQL Server, PostgreSQL, MySQL, Oracle.
