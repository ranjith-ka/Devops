## 29. Indexes

[▶ Watch this section](https://www.youtube.com/watch?v=btZdKO17xOM&t=15819s) · [⬆ Back to top](#table-of-contents)

Speed up queries by creating lookup structures on columns.

```sql
CREATE INDEX idx_emp_dept ON employees(dept_id);
```

Trade-off: faster reads, slightly slower writes.
