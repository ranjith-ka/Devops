## 12. CHECK Constraint

[▶ Watch this section](https://www.youtube.com/watch?v=btZdKO17xOM&t=4954s) · [⬆ Back to top](#table-of-contents)

Enforces a condition on column values.

```sql
CREATE TABLE employees (
    id INT PRIMARY KEY,
    age INT CHECK (age >= 18)
);
```
